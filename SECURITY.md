# Security Policy

## Supported versions

PrismForge follows active support for the latest release line:

- `latest` dist-tag: supported
- `next` dist-tag: best effort (pre-release)

## Reporting a vulnerability

Please do not report security vulnerabilities in public issues.

Use one of these private channels:

1. GitHub Security Advisory ("Report a vulnerability" in this repository)
2. Maintainer direct contact via repository security advisory discussion

Include:

- affected package(s) and version(s)
- reproduction steps or proof-of-concept
- expected vs actual behavior
- potential impact assessment

## Response targets

- Initial triage response: within 3 business days
- Ongoing updates: at least every 7 days until resolution

## Disclosure

After a fix is available, maintainers will coordinate responsible disclosure and publish remediation guidance in release notes.
