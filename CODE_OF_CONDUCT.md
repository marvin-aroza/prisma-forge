# Code of Conduct

## Our Commitment

PrismForge is committed to providing a welcoming, inclusive, and harassment-free community for everyone.

We expect all participants to:

- be respectful and constructive
- focus feedback on ideas and implementation details, not people
- assume good intent while holding a high bar for technical rigor
- help maintain a safe environment for contributors and users

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, discrimination, or personal attacks
- abusive language, intimidation, or threats
- publishing private information without explicit permission
- repeated disruptive behavior in issues, discussions, or pull requests

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. Maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this policy.

## Reporting

To report a violation, contact maintainers at:

- GitHub issue (if safe/public): `Security/Conduct` label
- private contact: open a GitHub Security Advisory draft for this repository

All reports will be reviewed promptly and handled confidentially where possible.

## Scope

This Code of Conduct applies within all project spaces and whenever an individual is officially representing the project.
